BREACH

Neo-Noir Glitch Platformer.

ABOUT THE GAME

Breach is a fast-paced arcade platformer inspired by vertical jumping games. Players collect and stack shells to fuel chained double jumps while navigating through glitch-filled neon circuit environments.

The game blends precision platforming mechanics with a cyberpunk aesthetic, challenging players to maintain momentum while navigating increasingly complex obstacles.

MY CONTRIBUTIONS

  • Designed and developed the entire game as a solo developer.
  • Programmed custom player controllers for precise movement and jumping mechanics.
  • Designed level progression and difficulty scaling.
  • Created the game's neo-noir glitch visual style and UI elements.
  • Implemented gameplay systems, effects, and UI using Unity.

DESIGN FOCUS

  • Precision platforming mechanics
  • Arcade gameplay progression
  • Cyberpunk glitch visual style
  • Solo game development workflow
